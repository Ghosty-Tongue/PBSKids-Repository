Admin Guide & Policies

1. Access to View Database

Access to the view database page is restricted to authorized personnel only. Do not share the URL https://openrbxl.com/pbskids/admin/view-database/ with unauthorized individuals. Leaking the URL may result in disciplinary action.

2. Data Handling

All data accessed through the view database page must be handled securely and in compliance with privacy regulations. Admins should only access data necessary for their duties and ensure that sensitive information is protected from unauthorized access or disclosure.
